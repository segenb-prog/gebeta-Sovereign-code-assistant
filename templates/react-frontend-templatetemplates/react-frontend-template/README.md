# React Frontend Template

A modern React template with TypeScript, routing, and API integration.

## Features

- React 18 with TypeScript
- React Router v6
- Axios for API calls
- Tailwind CSS
- Context API for state
- Unit tests with Jest
- Docker support

## Quick Start

```bash
# Create new app from template
cp -r templates/react-frontend-template /path/to/your-new-app

# Install dependencies
cd /path/to/your-new-app
npm install

# Start development server
npm run dev
```
